# 📊 Interactive Sales Dashboard (Excel)

## 📌 Project Overview

This project is an **Interactive Sales Dashboard** developed using **Microsoft Excel**. It provides valuable business insights by analyzing sales performance across different branches, cities, product lines, payment methods, and customer types.

The dashboard is fully interactive with slicers, allowing users to filter data dynamically and make better business decisions.

---

## 🛠️ Tools & Technologies Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Transformation
- Dashboard Design

---

## ✨ Dashboard Features

- 📈 Total Sales KPI
- 💰 Total Tax KPI
- 📦 Total Quantity KPI
- 💵 Unit Price KPI
- 🏢 Branch-wise Sales Analysis
- 🛍️ Product Line Analysis
- 🌍 City-wise Sales Analysis
- 💳 Payment Method Analysis
- 👥 Customer Type Analysis
- 🎛️ Interactive Slicers for Dynamic Filtering

---

## 📊 Key Insights

- Compared sales performance across Branch A, Branch B, and Branch C.
- Analyzed sales distribution by Product Line.
- Compared Member and Normal customer sales.
- Evaluated sales based on different Payment Methods.
- Compared sales across different Cities.
- Built an interactive dashboard with dynamic filters for easy analysis.
